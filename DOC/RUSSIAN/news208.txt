DN/2 2.08 alpha          Что нового

[!] Введена новая нефайловая панель - просмотр описаний. Вызов - Ctrl-D.
    Работает для всех файловых панелей, кроме архивов.
[!] Поддержка редактирования значений переменных окружения длиной
    более 255 символов (в текущей реализации до 16383 символов)
    
[*] Улучшен интерфейс диалога настроек страны
[-] После групповых операциях над файлами информация о числе выделенных
    файлов в подвале панели иногда не соответствовала выделению в панели.
    Ошибка внесена в beta 16 патчем jo30922b.dif.
[-] Под Win XP после команды cd с буквой диска на нижнем регистре возникали
    проблемы с перемещением по полосе дисков.
[-] Исправлено много ошибок в связи с копированием описаний.
[-] В просмотре dbf регистронезависимый поиск не работал.
    Ошибка внесена в 2.07 патчем ak41230a.dif.
[-] При переименовании каталога по F6 курсор не перемещался
    на переименованный файл.
[*] В диалог Расширенного фильтра добавлена строка ввода с текущей маской
[-] В файловых диалогах (скажем, по Shift-F4) скрытые каталоги не
    показывались, даже если включён показ скрытых файлов.

Заимствования из DN OSP 16 bit

[*] При попытке копирования/удаления недоступных файлов выдаётся запрос
    "Повторить/Пропустить/Пропустить все/Отменить".
[*] улучшено горизонтальное скроллирование в командной строке и редакторе
[*] При смене диска по Ctrl-стрелка на тот же самый, сохраняется позиция
    курсора.
