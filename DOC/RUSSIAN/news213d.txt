Основные отличия версии 2.13d devel от 2.13 alpha

[-] (не для новостей) При сборке DPMI32 не там формировались ресурсы
[-] (не для новостей) В просмотре исправлено и доделано следующее:
    - отображаются и вводятся не 8, а 9 Hex-цифр адреса (то есть до 64Г);
    - исправлены падения при "длинных" переходах;
    - закладки теперь имеют Y-составляющую типа TFileSize, так что
      совместимости по истории ввода со старыми версиями нет.
[*] (не для новостей) Теперь Hex8 строит буквы на верхнем регистре.
24.05.2007

[-] (не для новостей) В просмотре доделан показ адреса в нижней строке
[-] (не для новостей) [Win32] Не работало копирование из системного
    буфера обмена. Это ошибка RTL VP 2.1 b279, перенесенная в DN/2
    в 2.13.
[-] (не для новостей)[DPMI32] DN падал при определении типа диска на
    сетевых, subst и программных дисках.
    Ошибка внесена в 2.13 патчем ak70404a.dif.
[*] [DPMI32] Добавлено распознавание программных дисков, то есть таких,
    о которых система сообщает, как о сетевых, но не показывает сетевой путь.
[-] (не для новостей) Редактор не подствечивал синтаксис, а то и падал.
    Причина - не удалён временный отладочный код в 2.13.
25.05.2007

[-] (не для новостей) Большие файлы в архиве rar обрабатывались неправильно.
[-] (не для новостей) Большие  rar-архивы обрабатывались неправильно.
[-] (не для новостей) Большие файлы в архиве 7z обрабатывались неправильно.
29.05.2007

[-] (Не для новостей) В Hex-просмотре фильтры 32-255 и 0-255 фактически

[-] (не для новостей) Поддержка больших файлов архивов и больших файлов
    в архивах zip, ace, tar.
[-] При входе в битый пустой zip-архив DN падал.

[*] (Не для новостей) Теперь диалог точности калькулятора не закрывается
    по OK при наличии ошибок в строках ввода.
[*] (Не для новостей) Извлечение из архива каталога через временный

[!] Добавлена возможность прерывать копирование, созраняя недописанный

    (если целевой файл меньше исходного). Хелпы пока не откорректированы.
[-] Обработка директорий в архиве TAR
[*] Поддержка bzip2
[*] Мелкие коррекции текстовых строк в настройках архиваторов
[-] Обработка архивов HPACK с директориями внутри
